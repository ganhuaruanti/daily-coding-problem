## Daily Coding Problem: Problem #596 [Medium]

Good morning! Here's your coding interview problem for today.

This problem was asked by Google.

Invert a binary tree.

For example, given the following tree:

    
        a
       / \
      b   c
     / \  /
    d   e f

should become:

      a
     / \
     c  b
     \  / \
      f e  d
